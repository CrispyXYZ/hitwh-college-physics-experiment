# hitwh-college-physics-experiment

HITwh 哈工大（威海）大学物理实验

## 简介

自从2025级开始，学校把部分大类专业的大物实验课放到大一，本文旨在记录本人对此实验课程的体验与评价（部分），
包括各个实验项目的操作难度、教师指导情况、报告撰写要点以及可能遇到的坑点，希望能为后续同学提供一定的参考

## 上课流程

上课前需要完成实验报告的预习部分和预考核，进入实验室需要在讲台刷脸（有可能需要将报告交给老师检查），然后到点开始上课做实验。
课上主要是记录数据，把记录好数据的报告交给老师检查并盖章。到点才能离开，课下写数据处理和课后题，完成后交到报告箱

## 关于绪论课

26年春大物实验的绪论课是去听课，雨课堂签到并随堂做几道题，课下有线上的50题的考试作业，最终绪论课成绩=签到 40 分 + 作业 60 分（折算）

## 实验评价

|实验名称（老师）|评价|详情|
|---|---|---|
|固体线胀系数测定（焦威严）|夯爆了|我做的第一个实验。上课开门见山，直接讲步骤并介绍仪器的使用。 课上操作简单，但是会需要换算数据，[这里有一个简单的半自动换算脚本](Guti/calc.py)。课后的数据处理简单，只需要画图并算出结果，工作量小，数据处理只写了三行。最后得分4+5|
|表面张力系数测定（李丽）|人上人|我做的第二个实验。老师上课是启发式教学，也就是从原理一步步开始讲。操作虽然步骤多，但是简单。数据处理有些繁琐，需要计算不确定度, 当然也有[快捷计算网站](https://www.calculatorultra.com/zh/tool/standard-uncertainty-calculator.html)。|
|示波器（刘一）|夯爆了|老师上课会带着大家认真讲操作。实际操作也很简单，不会的找老师，老师会认真解答。数据处理偏简单。|
