# hitwh-college-physics-experiment

HITwh 哈工大（威海）大学物理实验

## 简介


大学物理实验是绝大多数专业都开设的独立实验课。本文旨在记录本人对该实验课程的体验与评价（不完全），涵盖各个实验项目的操作难度和注意事项等，希望能为后续同学提供一定的参考。

**当然最主要的还是数据处理脚本**

## 上课流程

上课前需要完成实验报告的预习部分和预考核，进入实验室需要在讲台刷脸，需要将报告交给老师检查（有的老师不会检查），然后到点开始上课做实验。

课上主要是记录数据，把记录好数据的报告交给老师检查并盖章。到点才能离开，课下写数据处理和课后题，完成后交到报告箱

## 关于绪论课

26年春大物实验的绪论课是去听课，雨课堂签到并随堂做几道题，课下有线上的50题的考试作业，最终绪论课成绩=签到 40 分 + 作业 60 分（折算）

## 实验评价

> 夯爆了>顶级>人上人>NPC>拉完了。注意同一个实验项目可能会有不同的老师教，可以去查查大物实验总课表。

|实验名称（老师）|综合评价|详情|
|---|---|---|
|固体线胀系数测定<br>（焦威严）|夯爆了|我做的第一个实验。上课开门见山，直接讲步骤并介绍仪器的使用。<br>课上操作简单，但是会需要换算数据，[这里有一个简单的半自动换算脚本](Guti/calc.py)。<br>课后的数据处理简单，只需要画图并算出结果，工作量小，数据处理只需要画图简单计算即可。<br>最后得分4+5|
|表面张力系数测定<br>（李丽）|顶级|我做的第二个实验。老师上课是从原理一步步开始讲。操作虽然步骤多，但是简单。<br>数据处理有些繁琐，需要计算不确定度, 当然也有[不确定度快捷计算网站](https://www.calculatorultra.com/zh/tool/standard-uncertainty-calculator.html)。<br>**缺点：此实验比较难计算**<br>得分4+4|
|示波器<br>（刘一）| ~~顶级~~ <br>夯爆了 |老师上课会带着大家认真讲操作。实际操作也很简单，不会的找老师，老师会认真解答。<br>数据处理偏简单。<br>得分5+5|
|偏振光<br>（员美娟）|顶级|老师很好。但是实验操作很累，要记录一百多组数据。<br>数据处理简单，直接上画图脚本：[直角坐标](Pianzhen/draw_1.py)，[极坐标](Pianzhen/draw_2.py)。<br>**缺点：此实验累颈椎**|
|动态磁滞回线<br>（范光华）|人上人|老师有点严格，会从实验原理开始讲，讲的一般。<br>建议课前认真预习，不过实验步骤是完全按照书上来操作的。<br>数据处理会有点麻烦，有[自动脚本](Dongtai/conv.py)，然后只需要画图<br>**缺点：此实验器材难以调整至合适**|
|等厚干涉<br>（王本阳）|顶级|老师很好，有不会的操作可以直接找老师帮你。<br>到时候建议上课前看看黑板的操作步骤，不要像我一样连着请了两次老师qwq。<br>数据处理会让算不确定度。<br>**缺点：此实验累眼睛累腰**|
